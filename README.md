# Python Scrapers 🕷️

A collection of my hands-on experiments and scripts (.ipynb) with Python for learning web scraping concepts, tools, and techniques.

## 📚 What's Inside

This repository documents my journey learning:
- **Web Scraping** - Extracting data from websites using Python libraries
- **Web Crawling** - Navigating and indexing multiple pages automatically
- **Data Parsing** - Cleaning and structuring scraped data for analysis

Each notebook represents a focused learning exercise with explanations and implementation.

## 🚀 Getting Started

Each notebook contains:

### Running a Notebook
```bash
# Clone the repository
git clone https://github.com/your-username/python-scrapers.git
cd python-scrapers

# Install dependencies
pip install -r requirements.txt

# Launch Jupyter
jupyter notebook
```
